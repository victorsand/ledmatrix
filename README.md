# LED Matrix message display and server

## Components
 - LED message display
 - Arduino microcontroller
 - Node.js server for serial communication and message queue API
 - Angular.js dashboard GUI

## Hardware
 - 32x16 LED matrix from Adafruit
 - Arduino Mega
 - Raspberry Pi

## Tech
 - C
 - JavaScript
 - Python
 - Bash
